# Elovyn Shopify Theme

A modern, premium Shopify theme for clothing and accessories stores.

## Installation

1. Zip the entire `theme` folder
2. In your Shopify admin, go to **Online Store > Themes**
3. Click **Add theme > Upload zip file**
4. Select your zipped theme folder
5. Click **Upload**

## Theme Structure

- `assets/` - CSS, JavaScript, and image files
- `config/` - Theme configuration (settings_schema.json)
- `layout/` - Main theme layout (theme.liquid)
- `sections/` - Reusable sections (header, hero, product showcases, footer)
- `snippets/` - Reusable code snippets
- `templates/` - Page templates (index.liquid)
- `locales/` - Translation files

## Sections

### Header
- Logo upload
- Navigation menu selection
- Mobile menu support
- Cart and account icons

### Hero
- Background image upload
- Customizable heading and subheading
- Call-to-action button
- Feature blocks at the bottom

### Clothing Showcase
- Displays products from a selected collection
- Configurable heading and description
- Number of products to display

### Accessories Showcase
- Same as Clothing Showcase but for accessories
- Displays products from a selected collection

### Hair Products Showcase
- Displays hair products from a selected collection

### Press Marquee
- Scrolling press quotes
- Add multiple press items

### Footer
- Logo upload
- Multiple link list columns
- Social media links
- Footer bottom menu

## Setup Instructions

1. **Create Collections**: Create collections for:
   - Clothing
   - Accessories
   - Hair Products

2. **Add Products**: Add your products to the appropriate collections

3. **Configure Sections**: 
   - Go to **Online Store > Themes > Customize**
   - Configure each section:
     - Upload logo in Header section
     - Select collections in showcase sections
     - Add press quotes in Press Marquee
     - Configure footer links

4. **Create Navigation Menus**:
   - Go to **Online Store > Navigation**
   - Create a main menu for the header
   - Create footer menus for footer links

5. **Upload Assets**: 
   - Upload your logo and hero image through the theme editor
   - Or upload directly to the `assets/` folder

## Customization

### Colors
- Customize colors in **Theme Settings > Colors**

### Typography
- Change fonts in **Theme Settings > Typography**

### Layout
- Adjust page width in **Theme Settings > Layout**
