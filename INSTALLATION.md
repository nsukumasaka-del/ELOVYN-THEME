# Shopify Theme Installation Guide

## Quick Start

1. **Zip the theme folder**
   - Navigate to the `theme` folder
   - Select all files and folders inside
   - Create a zip file named `elovyn-theme.zip`

2. **Upload to Shopify**
   - Log in to your Shopify admin
   - Go to **Online Store > Themes**
   - Click **Add theme**
   - Select **Upload zip file**
   - Choose your `elovyn-theme.zip` file
   - Wait for upload to complete

3. **Publish the theme**
   - Click **Actions > Publish** on your new theme

## Initial Setup

### 1. Create Collections

Create the following collections in your Shopify admin:

- **Clothing** - For clothing products
- **Accessories** - For accessories
- **Hair Products** - For hair care products

**Steps:**
- Go to **Products > Collections**
- Click **Create collection**
- Name your collection
- Add products to each collection

### 2. Create Navigation Menus

**Main Menu (Header):**
- Go to **Online Store > Navigation**
- Create a menu named "Main Menu" (or use existing)
- Add links to:
  - Clothing (#clothing)
  - Hair Products (#bikes)
  - Accessories (#features)
  - Other pages as needed

**Footer Menus:**
- Create separate menus for footer columns:
  - Clothing links
  - Accessories links
  - Company links
  - Support links

### 3. Customize the Theme

1. Go to **Online Store > Themes**
2. Click **Customize** on your theme
3. Configure each section:

**Header:**
- Upload your logo
- Select your main menu

**Hero:**
- Upload hero background image
- Customize heading and subheading
- Set button text and link
- Add feature blocks

**Product Showcases:**
- Select collections for each showcase
- Set number of products to display
- Customize headings and descriptions

**Press Marquee:**
- Add press quotes and publication names

**Footer:**
- Upload footer logo
- Add description
- Select footer menus
- Enable/disable social icons

### 4. Theme Settings

Go to **Theme Settings** to configure:

- **Colors**: Background, foreground, accent colors
- **Typography**: Fonts for headings and body text
- **Layout**: Page width
- **Social Media**: Add your social media links
- **Favicon**: Upload your favicon

## Product Setup

### Adding Products

1. Go to **Products > Add product**
2. Fill in product details:
   - Title
   - Description
   - Images (at least one featured image)
   - Price
   - Variants (if applicable)
3. Add product to appropriate collection(s)
4. Add vendor name (used as tagline in product cards)

### Product Images

- Recommended size: 1200x1500px (4:5 aspect ratio)
- Use high-quality images
- First image will be the featured image shown in showcases

## Testing

Before going live:

1. Test all navigation links
2. Verify product collections display correctly
3. Test mobile menu functionality
4. Check all buttons and links work
5. Test cart functionality
6. Verify responsive design on different devices

## Support

If you encounter issues:

1. Check that collections are created and have products
2. Verify navigation menus are set up correctly
3. Ensure all images are uploaded
4. Check browser console for JavaScript errors
