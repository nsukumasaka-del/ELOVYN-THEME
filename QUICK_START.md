# Quick Start Guide

Get your Shopify theme up and running quickly!

## Step 1: Upload Theme

1. Zip the entire `theme` folder
2. In Shopify Admin: **Online Store > Themes > Add theme > Upload zip file**
3. Select your zip file and wait for upload
4. Click **Publish** to make it live

## Step 2: Create Collections

Create these 3 collections in **Products > Collections**:

- **Clothing** (4 products)
- **Accessories** (4 products)  
- **Hair Products** (3 products)

See `COLLECTIONS_SETUP.md` for detailed instructions.

## Step 3: Add Products

Add products to match your current site. See `PRODUCTS_SETUP.md` for complete product list with:
- Product names
- Descriptions
- Prices (in ZAR)
- Vendor names (used as taglines)
- Image file names
- Purchase links

**Quick tip**: Use the **Vendor** field for the tagline that appears on product cards.

## Step 4: Upload Product Images

Copy images from `src/assets/` to your products:

- `Shirt 1.jpg` → Premium Shirt
- `Shirt 2.jpg` → Premium Shirt 2
- `Two Piece.jpg` → Summer Dress
- `CLOTHING.jpg` → Casual Hoodie
- `WEAVE 1.jpg` → Bundles
- `Shampoo 1.jpg` → Shampoo
- `HAND BAG COMBO.jpg` → 5‑in‑1 Hair Styling Tool

## Step 5: Configure Theme

1. Go to **Online Store > Themes > Customize**
2. Configure each section:

   **Header:**
   - Upload your logo (use `ELOVYN THE FINAL.png`)
   - Select your main navigation menu

   **Hero:**
   - Upload hero image (use `Front Page.png`)
   - Heading: "Modern Looks."
   - Subheading: "Timeless feel."
   - Button text: "Shop Now"

   **Clothing Showcase:**
   - Select "Clothing" collection
   - Products to show: 4

   **Accessories Showcase:**
   - Select "Accessories" collection
   - Products to show: 4

   **Hair Products Showcase:**
   - Select "Hair Products" collection
   - Products to show: 3

   **Footer:**
   - Upload logo
   - Add description
   - Configure footer menus

## Step 6: Create Navigation Menus

In **Online Store > Navigation**, create:

**Main Menu:**
- Clothes → #clothing
- Hair Products → #bikes
- Accessories → #features

**Footer Menus:**
- Create separate menus for each footer column

## Step 7: Test Everything

- ✅ Check all product images display correctly
- ✅ Verify collections show the right products
- ✅ Test mobile menu
- ✅ Check all links work
- ✅ Verify prices display correctly (ZAR format)
- ✅ Test cart functionality

## Important Notes

1. **Vendor = Tagline**: The product vendor field appears as the tagline on product cards
2. **Compare at Price**: Use for sale prices (e.g., Summer Dress shows 50% off)
3. **Product Images**: Recommended size 1200x1500px (4:5 aspect ratio)
4. **Currency**: Make sure ZAR (South African Rand) is set as your store currency
